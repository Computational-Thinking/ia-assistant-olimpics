    # Objetivos

    Este repositorio y este artículo contienen el material, informe y conclusiones de una investigación que nos planteamos los autores sobre el uso de los asistentes IA en la enseñanza de la programación.

    Nuestro objetivo es evaluar y comparar las capacidades de los diferentes asistentes IA para proveer soluciones a problemas como los que se plantean en la enseñanza de la programación.

    # Asistentes IA utilizados

    Decidimos usar todos los asistentes IA a nuestro alcance: Chat-GPT 3 y 4 y GitHub Copilot. Nos hubiera gustado probar GitHub Copilot X pero en el momento de hacer el estudio no tenemos acceso al mismo.

    # Problema

    Para medir las capacidades de los asistentes IA planteamos un problema de las Olimpiadas de Informática Española. El problema elegido ha sido el P31958_es que hemos tomado de jutge.org (opens new window)
